# Быстрый деплой на Vercel

## Вариант 1: Через GitHub (рекомендуется)
1. Перейти на https://vercel.com
2. Войти через GitHub
3. Нажать "New Project"
4. Выбрать репозиторий `heihezdrav-global-care`
5. Нажать "Deploy"

## Вариант 2: Через Vercel CLI
```bash
npm i -g vercel
vercel --prod
```

## Вариант 3: Через Netlify
1. Перейти на https://netlify.com
2. Drag & Drop папку `dist` после `npm run build`

Сайт будет доступен через 2-3 минуты!
